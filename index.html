Introduction

This website has been created to provide help and tutorials. The scope of tutorials and help includes Python and setting-up its environment in Ubuntu. If you are interested in learning how to use the Tkinter module in Python. I encourage you to follow the tutorial shown in Dinomuncher.

Dinomuncher shows some of the capabilities of Tkinter, which can be used to create a simple GUI for a game. If you are a true beginner I encourage you to thoroughly review the Requirements. Requirements also contains a basic tutorial on setting up a Python environment in Ubuntu.
